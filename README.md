# Metaog
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'metaog' , git: "git://github.com/tkyshm/metaog.git"
```

## Usage

```ruby

meta = Metaog.new "http://onepunchman-anime.net/"

meta.title

meta.url

meta.type

meta.image

meta.site_name

meta.description

```
